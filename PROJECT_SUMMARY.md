# Resumo do Projeto DonAssistec

## 📊 Estatísticas do Projeto

- **Versão**: 1.2.0
- **Data de Lançamento**: Janeiro 2026
- **Status**: ✅ Produção

## 🎯 Visão Geral

DonAssistec é uma plataforma B2B completa para gerenciamento de catálogo de peças para celulares, pedidos, serviços e relacionamento com lojistas e assistências técnicas.

- ✅ Dashboard completo com métricas e gráficos
- ✅ Gerenciamento de marcas e modelos
- ✅ Gerenciamento de serviços dinâmicos
- ✅ Gestão de pedidos e status
- ✅ Pré-pedidos (formulário de contato, urgente, exportação CSV)
- ✅ Gerenciamento de lojistas
- ✅ Sistema de tickets e suporte
- ✅ Relatórios e exportação (PDF, Excel, TXT)

## 🛠 Stack Tecnológico

### Frontend
- React 18 + TypeScript
- Vite
- Shadcn UI
- Tailwind CSS
- React Router DOM
- TanStack Query
- Socket.IO Client
- Recharts

### Backend
- Node.js + Express
- TypeScript
- MySQL (Docker)
- Socket.IO
- JWT
- Multer

### Infraestrutura
- Docker & Docker Compose
- PM2
- Nginx

## 📦 Módulos Principais

1. **Autenticação e Autorização**
   - JWT tokens
   - Role-based access control
   - Logs de auditoria

2. **Catálogo de Produtos**
   - CRUD de marcas
   - CRUD de modelos
   - Upload de imagens/vídeos
   - Serviços dinâmicos

3. **Sistema de Pedidos**
   - Criação e gerenciamento
   - Múltiplos status
   - Histórico completo
   - Exportação

4. **Configurações**
   - Branding e logo
   - Informações da empresa
   - Contato e mídias sociais
   - Configurações avançadas

5. **Analytics**
   - Dashboard com métricas
   - Gráficos e visualizações
   - Relatórios exportáveis
